babel-preset-flow [![version-i][]][npm]
========
It contains `syntax-flow` and `transform-flow-strip-types` plugins.

```js
// index.js
module.exports = {
  plugins: [
    require('babel-plugin-syntax-flow'),
    require('babel-plugin-transform-flow-strip-types'),
  ],
}
```
